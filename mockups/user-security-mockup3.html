User Security Help
How to use this interface:
  • Select User: Choose a user from the dropdown to view/edit their permissions
  • Toggle Permissions: Use the switches in the Access column to grant/deny permissions
  • Bulk Actions: Select multiple permissions using checkboxes, then use bulk action buttons
  • Search & Filter: Use the search box above the table to find specific permissions
  • Role Templates: Apply predefined permission sets for common roles
  • Save Changes: Click the Save button to persist all permission changes
